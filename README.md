# lightOS

A simple **terminal-based OS simulation** in Python.

## Features
help            - Show commands
 list            - List files
 open <file>     - Open text file
 calc            - Calculator
 google          - Google search
 myip            - Show your public IP
 update          - Update lightOS
 color           - Change color theme
 clear           - Clear screen
 exit            - Exit lightOS

## Requirements
- Python 3.x (no extra modules required)

## How to run

Clone the repository:

```bash
git clone https://github.com/matejtoseski-lang/lightOS.git
cd lightOS ls
python lightOS.py
